# CoreTech Solutions Website - Software House Website

## Project Overview
CoreTech Solutions is a responsive Software House Website developed using HTML, CSS, and JavaScript. It presents company services, portfolio projects, customer success stories, and contact information through a modern and user-friendly design.

## Features
- Responsive Design
- Modern Hero Section
- About Us Section
- Services Showcase
- Why Choose Us Section
- Portfolio Projects
- Customer Success Stories
- Contact Form Validation
- Mobile Navigation Menu

## Technologies Used
- HTML5
- CSS3
- JavaScript
