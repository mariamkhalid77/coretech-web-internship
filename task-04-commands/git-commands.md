# Git Commands

## 1. git init
Creates a new Git repository.

## 2. git clone <url>
Copies an existing repository.

## 3. git status
Shows repository status.

## 4. git add .
Stages all changes.

## 5. git commit -m "message"
Saves changes.

## 6. git push
Uploads changes to GitHub.

## 7. git pull
Downloads latest changes.

## 8. git branch
Shows branches.

## 9. git checkout branch-name
Switches branch.

## 10. git log
Shows commit history.
