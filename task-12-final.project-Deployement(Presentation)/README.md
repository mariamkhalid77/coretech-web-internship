# CoreTech Solutions - Software House Website

## Project Overview

CoreTech Solutions is a modern, responsive software house website developed as the final project for the Web Development Internship. The website showcases a professional business presence with multiple sections, interactive features, and a responsive design for all screen sizes.

---

## Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and professional user interface
- Mobile navigation menu
- Smooth scrolling navigation
- Active navigation highlighting
- Contact form with JavaScript validation
- LocalStorage integration for form data
- Scroll-to-top button
- Customer Success Stories section
- Portfolio section
- Responsive footer

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## Website Sections

- Home
- About Us
- Services

- Customer Success Stories
- Contact
- Footer

## JavaScript Functionality

- Responsive Mobile Menu

- Active Navigation Highlight
- Scroll-to-Top Button

## Installation

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any modern web browser.
